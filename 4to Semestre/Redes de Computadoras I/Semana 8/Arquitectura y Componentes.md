## Arquitectura de Red

- [I] **BSS (Basic Service Set):** Es la unidad fundamental de una red WiFi - 802.11. Consiste en estaciones inalámbricas que se comunican entre sí. Puede operar en modo de infraestructura (con un Punto de Acceso o AP) o en modo ad-hoc (sin AP).
- [I] **ESS (Extended Service Set):** Es una estructura de mayor escala formada por múltiples BSS conectados a través de un **Sistema de Distribución** (usualmente Ethernet). Esto permite que los usuarios se desplacen entre diferentes puntos de acceso sin perder la conectividad.
- [I] **SSID (Service Set Identifier):** Es el identificador de texto (nombre) que distingue a una red inalámbrica de otras.

## Componentes y Elementos de Identificación

- [I] **Estaciones (STA):** Son los dispositivos finales con capacidad inalámbrica, como computadoras portátiles, smartphones o tabletas.
- [I] **AP (Access Point; Punto de Acceso):** Dispositivo que actúa como puente entre la red inalámbrica y la red cableada, gestionando la actividad dentro de su célula.
- [I] **SSID (Service Set Identifier; Identificador de Conjunto de Servicio):** Es el nombre lógico de la red inalámbrica. Este identificador permite que las estaciones reconozcan y se asocien al conjunto de servicio correcto.

## Modos de Operación

La arquitectura permite dos formas principales de organizar la comunicación:

- **Modo Infraestructura:** Es el más común y utiliza un **AP** (Access Point; Punto de Acceso) central. En este modo, todas las estaciones se asocian al **AP** (Access Point; Punto de Acceso) y este retransmite las tramas entre ellas o hacia otras redes.
- **Modo Ad hoc:** En esta configuración, las estaciones se comunican directamente entre sí sin la intervención de un **AP** (Access Point; Punto de Acceso). Es una colección de ordenadores que se asocian de forma temporal y espontánea.
