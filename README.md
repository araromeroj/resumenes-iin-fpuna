# resumenes-iin-fpuna

¡Bienvenido/a a mi bóveda de apuntes personales! Este repositorio está sincronizado directamente con Obsidian y funciona como mi base de conocimiento (Zettelkasten/Cerebro Digital).

Para mantener la información estructurada, interconectada y fácil de consultar mediante automatizaciones (específicamente con el plugin **Dataview**), es necesario seguir unas reglas básicas al momento de crear y editar notas.

## Reglas de Creación de Notas

Para garantizar que la bóveda se mantenga ordenada y que el grafo de conocimiento sea útil, por favor sigue estos tres lineamientos clave:

### 1. Interconexión Obligatoria (Enlaces)

Toda nota nueva **debe estar conectada**.

- Al crear una nota, asegúrate de incluir enlaces (`[[Nombre del concepto]]`) hacia los temas, conceptos o índices relacionados con la información que estás escribiendo.
    
- Esto evita la creación de "notas huérfanas" y fortalece la red de conocimiento del repositorio.

### 2. Registro de Conceptos y Definiciones

Para registrar un concepto nuevo o una definición importante dentro de cualquier nota, debes usar un formato de tarea personalizada con la letra **`I`** (Information/Idea):

```Markdown
- [I] **Concepto:** Aquí va la definición del concepto.
```

> **¿Por qué hacer esto?** > Al usar este formato exacto, el plugin Dataview puede rastrear toda la bóveda y recopilar automáticamente todos los conceptos en una nota global, creando un diccionario o glosario dinámico sin esfuerzo manual.

### 3. Registro de Siglas y Acrónimos

De manera similar, cuando necesites definir el significado de una sigla o acrónimo, utiliza el formato de tarea personalizada con un asterisco **`*`**:

```Markdown
- [*] **ACRÓNIMO:** Significado del acrónimo.
```

> **¿Por qué hacer esto?** > Esto permite tener otra consulta de Dataview en una nota dedicada exclusivamente a recopilar todas las siglas mencionadas en cualquier parte de la bóveda, sirviendo como un índice automático de terminología.

- [i] Si tienes el tema de la bóveda "Tokyo Night" las viñetas o las tareas personalizadas tendrán símbolos. Con la letra I tendrás un foco y con el * tendrás una estrella. Puedes consultas las demás viñetas en la carpeta "Atajos y más de Obsidian".
- [i] Hay plantillas predeterminadas para tener estos accesos rápidos con el plugin de **dataview**. Puedes consultarlos en la carpeta "Recursos/Plantillas"
